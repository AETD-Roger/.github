# .github

Default community health files for all AETD-Roger repositories.

Any repository under this account that doesn't define its own copy automatically uses:

- [Pull request template](PULL_REQUEST_TEMPLATE.md)
- [Issue templates](ISSUE_TEMPLATE/) — bug report and feature request
- [Security policy](SECURITY.md)

A repository can override any of these by adding its own file of the same type.
